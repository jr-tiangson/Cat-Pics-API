# TheCatAPI — Cat Pics

This is a simple web project that lets you find cat images and information about different breeds using [TheCatAPI](https://thecatapi.com).  

It's built using only HTML, CSS, and JavaScript.

---

## Features

- Search cat breeds by name.
- Select a breed from a dropdown to see a cat image and info.
- Random cat image button.
- Dark mode toggle.
- Loading spinner while fetching data.
- Error messages for no results, invalid input, or failed API calls.
- Fully responsive design for any screen size.

---

## Project Files

- `index.html` — Main webpage.
- `style.css` — Styles for layout, cards, and buttons.
- `script.js` — All JavaScript code for API calls, DOM manipulation, and interactions.
